# Seminário sobre Processadores

## 📌 Sobre o Projeto

Este repositório foi criado para organizar e documentar o trabalho acadêmico de apresentação de um **seminário sobre processadores**, abordando conceitos fundamentais, arquitetura interna, funcionamento e evolução dessas unidades centrais de processamento.

O objetivo do seminário é apresentar de forma clara e técnica como os processadores funcionam, sua importância nos sistemas computacionais modernos e comparar modelos atuais do mercado.


## 🎯 Objetivos do Seminário

- Explicar o que é um processador e sua função no computador;
- Apresentar a estrutura interna de uma CPU;
- Demonstrar conceitos como núcleos, threads, clock e cache;
- Comparar fabricantes e modelos (Intel / AMD);
- Mostrar evolução histórica dos processadores;
- Relacionar desempenho com aplicações reais;
- Desenvolver habilidades de pesquisa e apresentação técnica.

### 5. Principais Fabricantes

### 6. Comparativos
- Desempenho
- Consumo energético

### 7. Tendências Futuras
- IA integrada
- Chips híbridos
- Eficiência energética
- Computação quântica
